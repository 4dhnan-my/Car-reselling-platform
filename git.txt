Step 1: Initialize Git in Your Project Folder:git init
Step 2: Add All Files :git add .
Step 3: Commit Your Files: git commit -m "Initial commit"
Step 4: Create a Repository on GitHub
Go to:
GitHub
Click New Repository
Give a name (e.g., ecommerce-django)
Click Create Repository
Step 5: Connect Local Repo to GitHub: git remote add origin https://github.com/yourusername/ecommerce-django. git
Step 6: Push Your Code:
git branch -M main                                                                                      
git push -u origin main
